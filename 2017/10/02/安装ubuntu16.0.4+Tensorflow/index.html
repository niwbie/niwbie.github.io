安装ubuntu16.0.4+Tensorflow

配置：
CPU ： E3-1230 v3 @3.30GHz
Mem: 2G
显卡: GTX1080

  1. 安装Ubuntu16.0.4操作系统
    操作系统这里选择的是64位Ubuntu16.0.4，建议安装桌面版。
    使用UNetbootin制作系统启动盘。制作过程很简单，不再多说。

  2. 开启ssh服务
    Ubuntu默认未安装ssh服务，因此需要先安装。
    apt-get install openssh-server
    sudo service sshd start

  3. 安装完后更新源
    修改 /etc/apt/sources.list
    deb http://mirrors.aliyun.com/ubuntu/ trusty main restricted universe multiverse
    deb http://mirrors.aliyun.com/ubuntu/ trusty-security main restricted universe multiverse
    deb http://mirrors.aliyun.com/ubuntu/ trusty-updates main restricted universe multiverse
    deb http://mirrors.aliyun.com/ubuntu/ trusty-proposed main restricted universe multiverse
    deb http://mirrors.aliyun.com/ubuntu/ trusty-backports main restricted universe multiverse
    deb-src http://mirrors.aliyun.com/ubuntu/ trusty main restricted universe multiverse
    deb-src http://mirrors.aliyun.com/ubuntu/ trusty-security main restricted universe multiverse
    deb-src http://mirrors.aliyun.com/ubuntu/ trusty-updates main restricted universe multiverse
    deb-src http://mirrors.aliyun.com/ubuntu/ trusty-proposed main restricted universe multiverse
    deb-src http://mirrors.aliyun.com/ubuntu/ trusty-backports main restricted universe multiverse

更新： sudo apt-get update
安装依赖 :sudo apt-get -f install
安装软件 :sudo apt-get install [software]

  1. 安装显卡驱动
    装完相应的依赖和软件之后，安装显卡驱动：

sudo add-apt-repository ppa:graphics-drivers/ppa
sudo apt-get update
sudo apt-get install nvidia-375
PS：如果系统使用UEFI启动，则可能安装不成功。所以先进主板，把UEFI启动关了！

ps:驱动卸载:
sh ./nvidia.run –uninstall

查看版本：
cat /proc/driver/nvidia/version

  1. 下载安装CUDA.
    sudo dpkg -i cuda-repo-ubuntu1604-8-0-local_8.0.44-1_amd64.deb
    sudo  apt-key adv --fetch-keys http://developer.download.nvidia.com/compute/cuda/repos/ubuntu1604/x86_64/7fa2af80.pub
sudo apt-get update 
sudo apt-get install cuda
    sudo apt-get install libcupti-dev

安装cuda时会出现：
Install NVIDIA Accelerated Graphics Driver for Linux-x86_64 384.81?这里要选择no，否则会重新安装显卡
确认 symlink正确设置：
scu@scu-B85M-D3V:~$ readlink -f /usr/local/cuda
/usr/local/cuda-9.0

测试一下
nvidia-smi:
有问题：

查看nvidia版本：

看来是版本太低了，于是卸掉已经装的驱动。。：
apt-get remove –purge nvidia-*
重启后查看：

  1. export PATH=/usr/local/cuda-8.0/bin${PATH:+:${PATH}}
  2. export LD_LIBRARY_PATH=/usr/local/cuda-8.0/lib64\
  3. export LD_LIBRARY_PATH=/usr/local/cuda-8.0/lib64${LD_LIBRARY_PATH:+:${LD_LIBRARY_PATH}}

进入/usr/local/cuda-8.0/cuda-install-samples-8.0.sh 安装Samples:

  1. 下载安装cuDNN(dev)
    https://developer.nvidia.com/rdp/cudnpythn-download

  2. 使用tar包
    解压后：
    sudo cp cuda/include/cudnn.h /usr/local/cuda/include
    sudo cp cuda/lib64/libcudnn /usr/local/cuda/lib64
    sudo chmod a+r /usr/local/cuda/include/cudnn.h /usr/local/cuda/lib64/libcudnn

  3. 使用deb包

2.1 安装runtime library：
sudo dpkg -i libcudnn7_7.0.3.11-1+cuda9.0_amd64.deb

2.2 安装developer library:

sudo dpkg -i libcudnn7-dev_7.0.3.11-1+cuda9.0_amd64.deb

2.3 安装code sample:

sudo dpkg -i libcudnn7-doc_7.0.3.11-1+cuda9.0_amd64.deb

然后进入刚刚CUDA安装的sample目录，默认是~/下，然后make，编译完成后输入
~/home/scu/cuda_sample\NVIDIA_CUDA-9.0_Samples/bin/x86_64/linux/release/deviceQuery

  1. 安装python环境
    依赖包
    sudo apt-get install python-pip python-dev build-essential python-numpy python-scipy python-matplotlib
  1. 安装anaconda
    bash Anaconda2-5.0.0-Linux-x86_64.sh

在安装的过程中，会询问安装路径，按回车即可。之后会询问是否将Anaconda安装路径（/root/Anaconda）加入到环境变量（.bashrc)中，输入yes，这样以后在终端中输入python即可直接进入Anaconda的Python版本（如果你的系统中之前安装过Python，自行选择yes or no）。安装成功后，会有当前用户根目录下生成一个anaconda2的文件夹，里面就是安装好的内容

设置环境变量
echo ‘export PATH=”~/anaconda2/bin:$PATH”‘ >> ~/.bashrc
source ~/.bashrc

查询安装信息：
conda info

查询当前已经安装的库

conda list

官方下载更新工具包的速度很慢，所以继续添加清华大学 TUNA提供的Anaconda仓库镜像，在终端或cmd中输入如下命令进行添加
conda config –add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config –set show_channel_urls yes
conda install numpy #测试是否添加成功

  1. 安装TensorFlow
    conda create -n tensorflow
    source activate tensorflow
    pip install –ignore-install –upgrade https://storage.googleapis.com/tensorflow/linux/gpu/tensorflow_gpu-1.3.0-cp27-none-linux_x86_64.whl

试一下：
python
import tensorflow

结果出错了：

tensorflow 1.3.0 -> cuda8.0 -> libcudnn.so.6

先把之前的删了：
pip uninstall tensorflow

然后运行pip install tensorflow
发现所有的pip安装程序都出错：

搜了半天没找到解决方法，猜测是pip版本问题。。。
sudo apt-get update –fix-missing
sudo apt-get autoremove && sudo apt-get clean && sudo apt-get install -f

然后尝试使用Anaconda。先到官网下载，

export TF_BINARY_URL=https://storage.googleapis.com/tensorflow/linux/gpu/tensorflow_gpu-1.3.0-cp27-none-linux_x86_64.whl

来自 https://www.tensorflow.org/install/install_linux#common_installation_problems
